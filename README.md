P a u s e. 

Welcome to the official website repository for P a u s e . 

P a u s e .  offers Yoga Nidra and relaxation sessions designed to help people pause, rest, and reconnect with themselves through guided relaxation and mindful rest.

About the Website

This website provides information about:

Yoga Nidra sessions
Upcoming events and classes
Booking information
Contact details

The website is intentionally designed to be calm, accessible, and easy to navigate.

Technology

This site is built using:

HTML5
CSS3
JavaScript
GitHub Pages

No frameworks are currently required, keeping the website lightweight, fast, and easy to maintain.

Development

To view the website locally:

Clone this repository.
Open the project folder in Visual Studio Code.
Open index.html in your browser or use the Live Server extension.
Deployment

The website is hosted using GitHub Pages.

Updates are published automatically when changes are pushed to the main branch.
